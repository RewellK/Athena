# CapabilityGapEngine

`CapabilityGapEngine` detecta quando a Athena não possui uma capacidade real para atender um pedido.

Tipos de lacuna:

- `missing_module`;
- `missing_source`;
- `missing_connector`;
- `missing_memory_capability`;
- `missing_research_strategy`;
- `missing_linguistic_pattern`;
- `missing_action_tool`;
- `missing_gui_interface`;
- `missing_validation_flow`.

O órgão não inventa resposta e não implementa código. Ele apenas registra a lacuna para que outro órgão proponha evolução segura.

Exemplo: pedido de preço de veículo sem fonte validada gera `missing_source` no domínio `vehicles`.
