# Cognitive Command Modes

A V13-pre inicia modos cognitivos controlados.

## Implementados

### aprendizagem

Observa, filtra e cria candidatos revisaveis.

Nao consolida automaticamente.

### status

Mostra runtime, pendencias, candidatos, insights e propostas locais.

Nao usa LLM.

### diagnostico

Mostra saude interna, heartbeat, ultimo erro, tarefas e buffer diario.

Nao executa reparo automatico perigoso.

### silencio

Pausa notificacoes e processos internos nao essenciais. A Athena continua respondendo quando chamada.

## Futuros

- memoria;
- reuniao;
- controle;
- foco;
- arquitetura.

Esses modos exigem escopo, politica, expiracao e aprovacao antes de qualquer poder adicional.
